# Amazon QuickSight — Features & Benefits Notes

---

## What Is Amazon QuickSight? (EXAM CRITICAL)

Amazon QuickSight is a:
- Fully managed
- Serverless
- Business intelligence (BI) service

It enables:
- Interactive dashboards
- Natural language queries
- Machine learning-powered insights
- Embedded analytics

---

## Core Purpose
QuickSight helps organizations:
- Understand data
- Explore dashboards
- Detect patterns
- Make data-driven decisions

---

# Key QuickSight Features

---

## 1️⃣ Connect and Scale All Your Data

### Connect to Multiple Sources
- AWS data sources (S3, Athena, Redshift, RDS)
- Third-party cloud providers
- On-premises data sources

---

### SPICE In-Memory Engine (EXAM FAVORITE)

SPICE:
- Super-fast, in-memory engine
- Scales to thousands of users
- Provides consistent performance
- Reduces load on underlying databases

---

### Data Modeling
- Combine multiple data sources
- Build governed data models
- Enable shared analytics across teams

---

## 2️⃣ Build Custom Dashboards

- Create use-case-specific dashboards
- Design interactive visualizations
- Provide personalized:
  - Email reports
  - Alerts

---

### Mobile Access
- iOS
- Android
- Mobile web

---

## 3️⃣ Machine Learning (ML) Integrations (EXAM CONCEPT)

QuickSight supports:

### Anomaly Detection
- Automatically detects:
  - Outliers
  - Variations

---

### Forecasting
- Predict business metrics
- What-if scenario analysis

---

### Auto-Narratives
- Automatically generate insights in plain language
- Provide contextual explanations in dashboards

---

## 4️⃣ Self-Service Business Intelligence

- Ask questions in natural language
- No deep BI expertise required
- Web-based visual analysis interface
- Embed analytics into applications

---

## 5️⃣ Native AWS Integration (EXAM FAVORITE)

### Secure Connectivity
- VPC connectivity
- Works with:
  - Redshift
  - Snowflake
  - RDS
  - Exasol

---

### IAM Integration
- Fine-grained access control
- Row-level security
- Column-level security

---

### SageMaker Integration
- Integrate ML models
- No complex pipelines required

---

## 6️⃣ Serverless & Cost Optimization

### No Infrastructure Management
- Fully managed
- No servers to provision

---

### Auto Scaling
- Scales to hundreds of thousands of users
- No overprovisioning required

---

### Pay-Per-Session Model (EXAM FAVORITE)
- Pay only for actual usage
- No need for large per-user license purchases
- Optimized for embedded analytics

---

## 7️⃣ Security & Compliance

### Encryption
- Encryption in transit
- Encryption at rest (including SPICE)

---

### Fine-Grained Security
- Row-level security
- Column-level security
- API support for automated governance

---

### Compliance Support
Supports compliance for:
- HIPAA
- GDPR
- SOC
- PCI
- ISO 27001
- FedRAMP High
- HITRUST
- And more
