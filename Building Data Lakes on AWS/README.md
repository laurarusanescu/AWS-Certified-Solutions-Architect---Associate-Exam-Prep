## Course Goal
- Explain the **value proposition of data lakes**
- Show **how to build, manage, and analyze data lakes on AWS**
- Combine:
  - Theory
  - AWS services
  - Hands-on labs

---

## Course Structure Overview

### Module 1: Introduction to Data Lakes
- What a **data lake** is
- Data lake vs **data warehouse**
- Why data lakes are important
- Core components and architectures

---

### Module 2: Data Ingestion, Cataloging, and Preparation
Focus on **getting data into the lake and making it usable**.

Key AWS services:
- **Amazon S3** – storage foundation
- **AWS Glue** – data cataloging & ETL
- **Amazon Athena** – query data in S3 using SQL
- Other ingestion and preparation services

---

### Module 3: AWS Lake Formation — Fundamentals
- Introduction to **AWS Lake Formation**
- Core characteristics and components
- Hands-on **self-paced lab**
  - Provisioned environment
  - Guided instructions

---

### Module 4: Data Processing and Analytics
What happens **after data is in the lake**.

Topics:
- Data transformation
- Data preparation

Key AWS services:
- **AWS Glue**
- **Glue DataBrew**
- **Glue Studio**

---

### Module 5: AWS Lake Formation — Advanced Features
Deeper dive into Lake Formation.

Topics include:
- Blueprints
- Workflows
- Fine-grained access control
- Tags
- Governance features

Additional service:
- **Amazon QuickSight**
  - Data visualization
  - Analytics dashboards

Includes:
- Self-paced lab

---

### Module 6: Modern Data Architectures
- Data movement patterns
- Data sharing models
- Modern data lake architectures

Includes:
- Self-paced lab

---

### Course Wrap-Up
- Key takeaways
- Next learning steps
- How to apply the knowledge
